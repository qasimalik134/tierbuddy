On the inside

Eco-Bikie is game built around an environment in which the main focus is to make a human life healthier. We are building a project in which we will be effectively moving to a concept in which the community can benefit by ride2earn functionality.

Users equip themselves with NFTs in the form of Bicycle. By riding outdoors and in gyms as well users will earn game currency, which can either be used in-game, or cashed out for profit.

With Game-Fi Eco-Bikie, aims to nudge millions toward a better and healthier lifestyle, combating climate changes and connect the public to Web 3.0, all while simultaneously hinging on its Social-Fi aspect to build a long-lasting platform fostering user generated Web 3.0 content
